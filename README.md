# Sablo v1 - Homepage

A modern, responsive homepage for Sablo Plywood built with React, Vite, and Tailwind CSS.

## Features

- ✅ Fully responsive design (mobile-first approach)
- ✅ Smooth animations using Framer Motion
- ✅ Parallax scrolling effects
- ✅ Counter animations
- ✅ Mobile hamburger menu with slide-in animation
- ✅ Scroll-triggered animations
- ✅ Performance optimized

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React (icons)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable components (Navbar, etc.)
├── sections/      # Page sections (Hero, About, Clients)
├── hooks/         # Custom React hooks
├── assets/        # Images and static assets
└── App.jsx        # Main app component
```
